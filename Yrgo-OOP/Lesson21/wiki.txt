<p><b>Enhanced Random Article</b> (<a href="/wiki/User:Gary/enhanced_random_article.js" title="User:Gary/enhanced random article.js">source</a>) is a script that adds a link above the "Random Article" link called "Enhanced Random Article". This script provides similar functionality to the built-in "Random Article" link, but there are three extra options, documented below.
</p>

<h3><span class="mw-headline" id="Custom_settings" data-mw-comment="{&quot;headingLevel&quot;:3,&quot;name&quot;:&quot;h-&quot;,&quot;type&quot;:&quot;heading&quot;,&quot;level&quot;:0,&quot;id&quot;:&quot;h-Custom_settings-Settings&quot;,&quot;replies&quot;:[]}"><span data-mw-comment-start="" id="h-Custom_settings-Settings"></span>Custom settings<span data-mw-comment-end="h-Custom_settings-Settings"></span></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikipedia:Enhanced_Random_Article&amp;action=edit&amp;section=4" title="Edit section: Custom settings">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>To change the default settings, copy the code above to <a href="/wiki/Special:MyPage/skin.js" title="Special:MyPage/skin.js">your custom JavaScript file</a> and change the values after the colons. The settings are explained in greater detail below. After changing the settings, <a href="/wiki/Wikipedia:BYPASS" class="mw-redirect" title="Wikipedia:BYPASS">bypass your browser's cache</a> or wait a while to see your changes take effect.
</p>
<dl><dt>patterns</dt>
<dd>The pattern that an article's title must match. It is case-sensitive. If more than one pattern is provided, then an article only needs to match one of the patterns.</dd>
<dd><b>Options:</b> This can be a string, such as <b><code class="mw-highlight mw-highlight-lang-text mw-content-ltr" id="" style="" dir="ltr">'List'</code></b> which would match article titles that contain "List"; or an array, such as <b><code class="mw-highlight mw-highlight-lang-text mw-content-ltr" id="" style="" dir="ltr">['a', 'e', 'i', 'o', 'u']</code></b> which would require that an article's title contain a vowel.</dd></dl>
<dl><dt>showDisambiguationPages</dt>
<dd>Set whether or not disambiguation pages are allowed to be shown.</dd>
<dd><b>Options:</b> Can be either <b><code class="mw-highlight mw-highlight-lang-text mw-content-ltr" id="" style="" dir="ltr">true</code></b> (to <i>allow</i> disambiguation pages) or <b><code class="mw-highlight mw-highlight-lang-text mw-content-ltr" id="" style="" dir="ltr">false</code></b> (to <i>disallow</i> disambiguation pages).</dd></dl>
<dl><dt>showStubs</dt>
<dd>Set whether or not stubs are allowed to be shown.</dd>
<dd><b>Options:</b> Can be either <b><code class="mw-highlight mw-highlight-lang-text mw-content-ltr" id="" style="" dir="ltr">true</code></b> (to <i>allow</i> stubs) or <b><code class="mw-highlight mw-highlight-lang-text mw-content-ltr" id="" style="" dir="ltr">false</code></b> (to <i>disallow</i> stubs).</dd></dl>
